Atlas of Living Australia Viz

Visualising the ALA

This page demonstrates a few techniques for visualising data from the Atlas of Living Australia. It uses d3js and twitter bootstrap.

Start by searching for a name in the search box above (something like Eucalyptus or Macropus works well).
