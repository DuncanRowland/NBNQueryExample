NBN Atlas Viz

Visualising the NBN Atlas

This page demonstrates a few techniques for visualising data from the NBN Atlas. It uses d3js and twitter bootstrap.

Start by searching for a name in the search box above (something like Fox works well). You will then be able to see the occurrences grouped according to the categories on the right. Clicking on any circle will run a new query rather than drill down further.

This page adapted from Peter Neish's original example for the Atlas of Living Australia.
